# Vehicle-to-Load-V2L-for-Powering-External-Loads

This repository contains a detailed project report on the design, modeling and evaluation of a Vehicle-to-Load (V2L) system built in MATLAB and Simulink.

**Contents**

Project_Report.pdf
Complete technical report covering system architecture, control strategies, converter design and simulation results.

**Project Overview**

The project focuses on designing a V2L system that allows an electric vehicle battery to supply power to external loads while maintaining safety, efficiency and battery longevity. The work includes control development, converter design and simulation studies.

**Key Features**

Designed a V2L system that supports bidirectional power flow for external loads and potential grid support.

Developed predictive and adaptive control algorithms in MATLAB and Simulink to manage energy transfer, optimize SOC and DOD, and minimize thermal stress.

Achieved improvements in overall energy efficiency through optimized control and converter operation.

Designed and validated a high-efficiency bidirectional DC-DC converter.

Tested system performance under realistic load scenarios and analyzed stability, efficiency and battery protection behavior.
